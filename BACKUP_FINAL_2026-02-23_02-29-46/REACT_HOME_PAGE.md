## 🔧 Использованные компоненты

### GlowingEffect
Основной компонент эффекта свечения:
```tsx
<GlowingEffect disabled={false} blur={10} spread={20} />
```

**Параметры:**
- `disabled` - включение/отключение эффекта
- `blur` - размытие эффекта
- `spread` - распространение градиента
- `proximity` - расстояние активации
- `inactiveZone` - неактивная зона в центре
- `variant` - вариант цвета (default/white)
- `borderWidth` - ширина границы
- `glow` - видимость свечения
- `movementDuration` - длительность анимации

### GlowingCard
Обертка для карточек с отслеживанием мыши:
```tsx
<GlowingCard>
  {/* Содержимое карточки */}
</GlowingCard>
```

**Особенности:**
- Автоматическое отслеживание движения мыши
- Интегрирован GlowingEffect
- Поддержка onClick
- Кастомные классы
