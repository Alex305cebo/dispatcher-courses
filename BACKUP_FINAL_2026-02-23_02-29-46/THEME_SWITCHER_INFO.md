# Переключатель Темы - Инструкция

## ✅ Что сделано

Добавлен универсальный переключатель темы (светлая/темная) на **все 42 страницы** проекта!

## 🎨 Особенности

### Темная тема (по умолчанию)
- Темно-синий фон (#070b14)
- Светлый текст (#f0f4ff)
- Фиолетово-синие акценты

### Светлая тема
- Светло-серый фон (#f5f7fa)
- Темный текст (#1a202c)
- Те же фиолетово-синие акценты для гармонии

## 🔘 Кнопка переключателя

- **Расположение**: Правый верхний угол (фиксированная позиция)
- **Иконки**: 
  - 🌙 - темная тема активна
  - ☀️ - светлая тема активна
- **Размер**: 50x50px (45x45px на мобильных)
- **Эффекты**: Увеличение при наведении, плавные переходы

## 💾 Сохранение настроек

Выбранная тема автоматически сохраняется в `localStorage` и применяется при следующем посещении любой страницы сайта.

## 📁 Файлы

- `theme-switcher.js` - Универсальный скрипт переключателя
- `add-theme-switcher.ps1` - Скрипт для автоматического добавления на все страницы
- `remove-theme-switcher.ps1` - Скрипт для удаления (если понадобится)

## 🔧 Добавление на новые страницы

Для новых HTML страниц добавьте перед закрывающим тегом `</body>`:

### Для страниц в корне:
```html
<script src="theme-switcher.js"></script>
</body>
```

### Для страниц в папке pages/:
```html
<script src="../theme-switcher.js"></script>
</body>
```

Или просто запустите скрипт:
```powershell
.\add-theme-switcher.ps1
```
