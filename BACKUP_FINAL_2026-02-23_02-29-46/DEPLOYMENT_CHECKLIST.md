## 📦 Нативное приложение (Для магазинов)

### Подготовка окружения

#### Для Android:
- [ ] Node.js 16+ установлен
- [ ] Android Studio установлен
- [ ] Android SDK настроен
- [ ] Java JDK установлен
- [ ] Переменные окружения настроены

#### Для iOS (только Mac):
- [ ] Xcode установлен
- [ ] Command Line Tools установлены
- [ ] CocoaPods установлен
- [ ] Apple Developer Account создан ($99/год)

### Установка Capacitor
```bash
- [ ] npm install @capacitor/core @capacitor/cli
- [ ] npm install @capacitor/android @capacitor/ios
- [ ] npx cap init "Диспетчер США" "com.dispatcher.courses"
- [ ] npx cap add android
- [ ] npx cap add ios
- [ ] npx cap sync
```

### Настройка Android

#### Конфигурация:
- [ ] android/app/build.gradle настроен
- [ ] Версия приложения установлена
- [ ] Минимальная SDK версия: 21
- [ ] Target SDK версия: 33
- [ ] Permissions добавлены

#### Подпись приложения:
- [ ] Keystore создан
```bash
keytool -genkey -v -keystore my-release-key.keystore \
  -alias my-key-alias -keyalg RSA -keysize 2048 -validity 10000
```
- [ ] Keystore пароль сохранен
- [ ] build.gradle обновлен с данными keystore

#### Сборка APK:
```bash
- [ ] cd android
- [ ] ./gradlew assembleRelease
- [ ] APK создан в: app/build/outputs/apk/release/
- [ ] APK протестирован на устройстве
```

### Настройка iOS

#### Конфигурация:
- [ ] Xcode проект открыт
- [ ] Bundle Identifier установлен
- [ ] Версия приложения установлена
- [ ] Deployment Target: iOS 13.0+
- [ ] Signing настроен

#### Сертификаты:
- [ ] Apple Developer Account активен
- [ ] Development Certificate создан
- [ ] Distribution Certificate создан
- [ ] Provisioning Profiles созданы
- [ ] Сертификаты установлены в Xcode

#### Сборка IPA:
```bash
- [ ] Product > Archive
- [ ] Архив создан успешно
- [ ] Validate App пройдена
- [ ] Distribute App выполнена
```
