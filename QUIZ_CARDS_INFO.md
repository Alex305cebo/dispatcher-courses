# 🎯 Интерактивные карточки вопросов (Quiz Cards)

## Описание

Интерактивное приложение для обучения диспетчеров в стиле популярного приложения Tinder. Карточки с вопросами можно свайпать влево (неверный ответ) или вправо (правильный ответ).

## Особенности

### 🎨 Дизайн
- Полноэкранные карточки с вопросами
- Плавные анимации свайпа
- Визуальные индикаторы правильных/неправильных ответов
- Адаптивный дизайн для всех устройств

### 📱 Мобильная адаптация
- Touch-события для свайпа на мобильных устройствах
- Mouse-события для работы в браузере
- Вибрация при ответах (на поддерживаемых устройствах)
- Оптимизация для iOS и Android

### 🎮 Интерактивность
- Свайп влево = неверный ответ (✗)
- Свайп вправо = правильный ответ (✓)
- Кнопки для тех, кто предпочитает клики
- Кнопка информации для показа подсказок

### 📊 Прогресс и статистика
- Отображение текущего вопроса
- Прогресс-бар выполнения
- Счётчик правильных/неправильных ответов
- Детальные результаты по завершении

## Структура файлов

```
quiz-cards.html     - Основная HTML-структура
quiz-cards.css      - Стили и анимации
quiz-cards.js       - Логика приложения и база вопросов
```

## База вопросов

Вопросы хранятся в массиве `questionsDatabase` в файле `quiz-cards.js`. Каждый вопрос содержит:

```javascript
{
    id: 1,                          // Уникальный ID
    category: "Основы работы",      // Категория вопроса
    question: "Текст вопроса?",     // Сам вопрос
    description: "Подробное описание", // Объяснение
    correctAnswer: true,            // Правильный ответ (true/false)
    hint: "Подсказка"              // Дополнительная подсказка
}
```

## Добавление новых вопросов

Чтобы добавить новые вопросы, отредактируйте массив `questionsDatabase` в `quiz-cards.js`:

```javascript
const questionsDatabase = [
    // ... существующие вопросы
    {
        id: 11,
        category: "Новая категория",
        question: "Ваш вопрос?",
        description: "Подробное объяснение ответа",
        correctAnswer: true,
        hint: "Полезная подсказка"
    }
];
```

## Функциональность

### Свайп-механика
- Карточка следует за пальцем/курсором
- При свайпе более 100px срабатывает ответ
- Визуальная обратная связь (индикаторы ✗ и ✓)
- Плавная анимация исчезновения карточки

### Кнопки управления
- **Красная кнопка (✗)** - Неверный ответ
- **Белая кнопка (ℹ)** - Показать/скрыть подсказку
- **Зелёная кнопка (✓)** - Правильный ответ

### Результаты
После прохождения всех вопросов показывается модальное окно с:
- Количеством правильных ответов
- Количеством неправильных ответов
- Процентом успешности
- Кнопками "Начать заново" и "На главную"

## Интеграция с платформой

Приложение интегрировано в личный кабинет (`dashboard.html`) как отдельная карточка курса "Тренажёр вопросов".

## Использование

1. Откройте `dashboard.html`
2. Найдите карточку "Тренажёр вопросов"
3. Нажмите "Начать тренировку"
4. Свайпайте карточки или используйте кнопки
5. Просматривайте результаты по завершении
