# 👤 Личный кабинет

## Что включено

Полноценный личный кабинет для зарегистрированных пользователей с:

### 📊 Главная панель
- Приветствие с именем пользователя
- Статистика обучения:
  - Активные курсы
  - Завершённые курсы
  - Часы обучения
  - Полученные сертификаты

### 📚 Мои курсы
- Список всех курсов пользователя
- Прогресс по каждому курсу (визуальный индикатор)
- Кнопки для продолжения обучения
- Статус завершения

### 👤 Профиль
- Аватар пользователя
- Личная информация (имя, фамилия, email)
- Статистика профиля
- Редактирование данных профиля
- Добавление телефона

### 🏆 Сертификаты
- Список полученных сертификатов
- Дата выдачи
- Кнопка скачивания PDF
- Заблокированные сертификаты (для незавершённых курсов)

### ⚙️ Настройки
- Смена пароля
- Настройки уведомлений:
  - Email о новых курсах
  - Напоминания о незавершённых уроках
  - Новости платформы

## Доступ к личному кабинету

### Для новых пользователей:
1. Зарегистрируйтесь на сайте (register.html)
2. После регистрации вы автоматически попадёте в личный кабинет

### Для существующих пользователей:
1. Войдите в систему (login.html)
2. После входа вы попадёте в личный кабинет
3. Или нажмите на своё имя в правом верхнем углу

### Прямая ссылка:
```
http://localhost:3000/dashboard.html
```

## Функции

### ✅ Работает сейчас:
- Проверка авторизации (редирект на login если не авторизован)
- Отображение данных пользователя
- Переключение между вкладками
- Редактирование профиля (сохранение в localStorage)
- Визуализация прогресса курсов
- Адаптивный дизайн

### 🔄 Будет добавлено:
- Реальные данные курсов из базы
- Система прогресса обучения
- Генерация PDF сертификатов
- Смена пароля через API
- История платежей
- Календарь вебинаров

## Навигация

В личном кабинете доступны 4 вкладки:

1. **Мои курсы** - активные и завершённые курсы
2. **Профиль** - личная информация и редактирование
3. **Сертификаты** - полученные сертификаты
4. **Настройки** - настройки аккаунта и уведомлений

## Дизайн

- Современный UI в стиле всего сайта
- Тёмная/светлая тема (наследуется)
- Градиенты и анимации
- Карточки с hover эффектами
- Адаптивная вёрстка для мобильных

## Безопасность

- Проверка JWT токена при загрузке
- Автоматический редирект на login если не авторизован
- Данные пользователя хранятся в localStorage
- Email нельзя изменить (защита от случайной смены)

## Интеграция

Личный кабинет интегрирован со всем сайтом:
- Ссылка в навигации (только для авторизованных)
- Клик на имя пользователя ведёт в кабинет
- После регистрации/входа - автоматический переход
- Кнопка "Выйти" доступна везде

## Расширение функционала

Для добавления новых функций в личный кабинет:

1. Добавьте новую вкладку в HTML
2. Создайте соответствующий tab-content
3. Добавьте обработчик в switchTab()
4. При необходимости добавьте API endpoint в server.js

## Примеры использования

### Обновление статистики:
```javascript
document.getElementById('activeCourses').textContent = '5';
document.getElementById('studyHours').textContent = '48';
```

### Добавление нового курса:
```javascript
const courseCard = `
  <div class="course-card">
    <div class="course-image">🎯</div>
    <div class="course-content">
      <h3 class="course-title">Новый курс</h3>
      <div class="course-progress">
        <div class="progress-label">
          <span>Прогресс</span>
          <span>0%</span>
        </div>
        <div class="progress-bar">
          <div class="progress-fill" style="width: 0%"></div>
        </div>
      </div>
      <button class="btn-continue">Начать →</button>
    </div>
  </div>
`;
document.querySelector('.courses-grid').innerHTML += courseCard;
```

## Поддержка

Если возникли вопросы или проблемы с личным кабинетом:
1. Проверьте, что вы авторизованы
2. Проверьте консоль браузера на ошибки
3. Убедитесь, что сервер запущен (npm start)
4. Очистите localStorage и войдите заново
