# Audio Files for Broker Responses

This folder contains audio files for broker responses in the call training dialogue.

## File Naming Convention

Files are named: `broker_response_1_[step].mp3`

Where `[step]` corresponds to the dialogue step number:

- `broker_response_1_1.mp3` - First broker response (after greeting)
- `broker_response_1_3.mp3` - Second broker response (after equipment confirmation)
- `broker_response_1_5.mp3` - Third broker response (after rate question)
- `broker_response_1_7.mp3` - Fourth broker response (after negotiation start)
- `broker_response_1_9.mp3` - Fifth broker response (after compromise offer)
- `broker_response_1_11.mp3` - Sixth broker response (final confirmation)

## How to Replace Audio Files

1. Record or obtain the audio files for each broker response
2. Convert them to MP3 format
3. Replace the corresponding file in this folder
4. Keep the same filename to ensure the dialogue works correctly

## Audio Requirements

- Format: MP3
- Recommended bitrate: 128 kbps or higher
- Language: English (broker responses)
- Duration: Keep responses concise (10-30 seconds each)

## Notes

- The audio icon (🔊) appears next to "Брокер" label in each broker response
- Clicking the icon will play the corresponding audio file
- Only one audio can play at a time
- The icon animates while audio is playing
