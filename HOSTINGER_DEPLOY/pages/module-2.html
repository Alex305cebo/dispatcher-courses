FMCSA и DOT регулирование

Нормативная база транспортной индустрии

📚 Содержание модуля

1. FMCSA - структура и функции

FMCSA (Federal Motor Carrier Safety Administration) - федеральное агентство, отвечающее за безопасность грузоперевозок.

  • Основные функции: Регулирование безопасности, лицензирование перевозчиков, контроль соблюдения правил
  • Safety Rating: Оценка безопасности перевозчика (Satisfactory, Conditional, Unsatisfactory)
  • CSA Program: Compliance, Safety, Accountability - система мониторинга безопасности
  • Enforcement: Проверки, штрафы, приостановка деятельности при нарушениях
  • SMS (Safety Measurement System): Система оценки безопасности по 7 категориям BASIC

2. DOT номера и регистрация

DOT Number - уникальный идентификационный номер перевозчика, необходимый для всех коммерческих перевозок.

  • Когда нужен: Если транспорт пересекает границы штатов или перевозит опасные грузы
  • Как получить: Регистрация через FMCSA Portal, заполнение формы MCS-150
  • Стоимость: $300 регистрационный сбор
  • Обновление: Каждые 2 года необходимо обновлять информацию
  • Проверка DOT: Через сайт FMCSA Safer System можно проверить данные перевозчика
  • Штрафы: DOT номер должен быть виден на обе стороны грузовика (высота цифр минимум 2 дюйма)

3. MC Authority и Operating Authority

MC Number (Motor Carrier) - разрешение на осуществление коммерческих перевозок за плату.

  • MC Authority: Разрешение для перевозчиков, работающих за плату
  • FF Authority: Разрешение для freight forwarders (экспедиторов)
  • Broker Authority: Разрешение для брокеров грузоперевозок
  • Процесс получения: Подача заявки, оплата $300, ожидание 20-30 дней
  • BOC-3: Обязательное назначение агентов по юридическому процессу в каждом штате
  • Insurance Filing: Предоставление подтверждения страхового покрытия (BMC-91 или BMC-91X)

4. Требования к водителям (CDL)

CDL (Commercial Driver's License) - коммерческие водительские права для управления коммерческим транспортом.

  • Class A CDL: Для тягачей с прицепами (комбинированный вес >26,001 lbs)
  • Class B CDL: Для одиночных грузовиков (вес >26,001 lbs)
  • Class C CDL: Для небольших грузовиков или пассажирских
  • Endorsements: H (HAZMAT), N (Tank), T (Double/Triple Trailers), P (Passengers)
  • Medical Certificate: Медицинская справка каждые 2 года
  • Drug Testing: Обязательное тестирование на наркотики (pre-employment, random, post-accident)
  • MVR (Motor Vehicle Record): История вождения водителя

5. Страхование и ответственность

Обязательное страхование для коммерческих перевозчиков:

  • Liability Insurance: Минимум $750,000 для общих грузов
  • HAZMAT Insurance: Минимум $5,000,000 для опасных грузов
  • Cargo Insurance: Рекомендуется минимум $100,000 (опционально, но часто требуется)
  • Physical Damage: Страхование самого транспорта и оборудования
  • Workers Compensation: Страхование работников (если есть сотрудники)
  • General Liability: Общая ответственность ($1-2 миллиона)
  • Важно: Страховка должна быть активна 24/7, иначе MC Authority может приостановиться

6. Штрафы и нарушения

Основные нарушения и штрафы:

  • HOS Violations: $1,000-$11,000 за нарушение правил рабочего времени
  • Overweight: $100-$10,000 за превышение веса
  • No Insurance: До $25,000 + приостановка деятельности
  • Unsafe Driving: $2,500-$5,000 за опасное вождение
  • False Logs: До $15,000 за фальсификацию логов
  • No Medical Card: $1,000-$2,500 за отсутствие медицинской карты
  • CSA Points: Накопление баллов может привести к проверкам и приостановке Authority
