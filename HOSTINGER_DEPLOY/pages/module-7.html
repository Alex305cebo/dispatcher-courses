Маршрутизация и планирование

Оптимизация маршрутов и контроль времени

📚 Содержание модуля

1. Инструменты маршрутизации

Популярные инструменты:

  • Google Maps: Бесплатный инструмент для базовой маршрутизации
  • PC Miler: Профессиональный инструмент для грузовиков ($50-100/месяц)
  • Trucker Path: Мобильное приложение с truck stops, parking, weigh stations
  • Waze: Навигация с учётом пробок и дорожных условий
  • CoPilot Truck GPS: Платная навигация для грузовиков

Что учитывать:

  • Truck Routes: Используйте маршруты для грузовиков, а не для легковых авто
  • Bridge Heights: Проверяйте высоту мостов (стандартный трак 13'6")
  • Weight Restrictions: Ограничения по весу на мостах и дорогах
  • Toll Roads: Платные дороги - учитывайте в расходах
  • Truck Stops: Планируйте остановки для отдыха, топлива, еды
  • Weigh Stations: Обязательные остановки на весовых

2. Расчёт времени в пути

  • Средняя скорость: 50-55 mph с учётом остановок (не 65-70 mph!)
  • Формула: Мили / 50 mph = Часы в пути
  • Пример: 500 миль / 50 mph = 10 часов
  • Добавьте время на: Завтрак (30 мин), обед (30 мин), топливо (15-30 мин)
  • 30-минутный перерыв: Обязательный после 8 часов вождения
  • Погрузка/разгрузка: 1-3 часа в зависимости от типа

3. Учёт часовых поясов

Основные часовые пояса США:

  • Eastern Time (ET): NY, FL, GA, NC, SC, VA, PA, OH, MI
  • Central Time (CT): TX, IL, MO, TN, AL, LA, MN, WI
  • Mountain Time (MT): CO, AZ, NM, UT, WY, MT, ID
  • Pacific Time (PT): CA, WA, OR, NV
  • Разница: ET > CT (-1 час), CT > MT (-1 час), MT > PT (-1 час)

4. Погодные условия

  • Снег и лёд: Снижает скорость на 30-50%, опасность аварий
  • Дождь: Снижает скорость на 10-20%, ухудшает видимость
  • Туман: Серьёзно ухудшает видимость
  • Ветер: Опасен для high-profile грузов
  • Жара: Риск перегрева двигателя, blow-outs шин

Инструменты проверки погоды:

  • Weather.com, NOAA Weather, Trucker Path, 511 Systems

5. Дорожные ограничения

  • Height Restrictions: Мосты, тоннели (стандарт 13'6")
  • Weight Limits: Максимальный вес 80,000 lbs
  • Length Restrictions: Некоторые штаты ограничивают длину
  • Hazmat Restrictions: Запрет на проезд через тоннели, мосты
  • No Truck Zones: Зоны, где грузовики запрещены
  • Parkways: Дороги, где коммерческий транспорт запрещён

6. Минимизация deadhead

  • Определение: Пустой пробег без груза (не приносит доход)
  • Backhaul Planning: Ищите обратный груз перед завершением текущего
  • Hot Lanes: Работайте на популярных маршрутах
  • Acceptable Deadhead: 50-100 миль приемлемо, >150 миль - плохо
  • Расчёт: Включайте deadhead в расчёт RPM
  • Пример: Груз $2,000 за 500 миль + 100 миль deadhead = $2,000 / 600 миль = $3.33/mile

  • Мониторинг погодных условий
