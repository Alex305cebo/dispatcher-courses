# 📱 Руководство по мобильной версии

## 🚀 Быстрый старт

### Вариант 1: Предпросмотр (рекомендуется)
1. Откройте файл `mobile-preview.html` в браузере
2. Выберите платформу:
   - **iOS Preview** - для просмотра версии iPhone
   - **Android Preview** - для просмотра версии Android
3. Все ссылки работают и ведут на реальные страницы

### Вариант 2: Прямой доступ
1. Откройте `index.html` на мобильном устройстве
2. Все страницы автоматически адаптируются
3. Навигация работает как на десктопе

### Вариант 3: Установка как приложение (PWA)

#### На Android:
1. Откройте сайт в Chrome
2. Нажмите меню (⋮) в правом верхнем углу
3. Выберите "Установить приложение" или "Добавить на главный экран"
4. Подтвердите установку
5. Приложение появится на главном экране

#### На iOS:
1. Откройте сайт в Safari
2. Нажмите кнопку "Поделиться" (квадрат со стрелкой)
3. Прокрутите вниз и выберите "На экран Домой"
4. Введите название (или оставьте по умолчанию)
5. Нажмите "Добавить"
6. Иконка появится на главном экране

### Система прогресса
- **Не пройдено** (красный) - тест еще не пройден
- **Пройдено: X%** (желтый) - результат менее 70%
- **Пройдено: X%** (зеленый) - успешно пройдено (≥70%)

## 💡 Советы по использованию

### Для лучшего опыта:
1. **Начните с документации** - это обязательный первый шаг
2. **Проходите модули по порядку** - материал построен последовательно

- Кнопка **"← Назад"** на каждой странице

После первого посещения приложение работает офлайн благодаря Service Worker

## 🆘 Решение проблем

### Приложение не устанавливается:
1. Убедитесь, что используете Chrome (Android) или Safari (iOS)
2. Проверьте подключение к интернету
3. Очистите кэш браузера
4. Попробуйте перезагрузить страницу

### Ссылки не работают:
1. Проверьте, что все файлы на месте
2. Убедитесь, что структура папок не изменена
3. Попробуйте открыть через локальный сервер

### Прогресс не сохраняется:
1. Проверьте, что localStorage включен в браузере
2. Не используйте режим инкогнито
3. Проверьте настройки приватности браузера

### Страница не адаптируется:
1. Проверьте viewport meta tag
2. Попробуйте обновить страницу
3. Очистите кэш браузера
