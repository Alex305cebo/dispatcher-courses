# ⏰ Что настроить через 8 часов

## 📅 Дата создания: 23 февраля 2024
## ⏰ Настроить после: ~8 часов (примерно в 23:00 или позже)

---

## 🔐 1. Включить HTTPS для dispatch4you.com

### Шаг 1: Открой GitHub Pages Settings
https://github.com/Alex305cebo/dispatcher-courses/settings/pages

### Шаг 2: Проверь статус SSL сертификата
Если видишь сообщение что сертификат готов:
- Поставь галочку **Enforce HTTPS**
- Нажми **Save**

### Шаг 3: Проверь сайт
Открой https://dispatch4you.com (с HTTPS)

---

## 🔗 2. Привязать dispatcher-cards-app к основному домену (опционально)

Если хочешь чтобы карточки были на dispatch4you.com/cards:

### Вариант A: Поддомен (cards.dispatch4you.com)
1. В Hostinger DNS добавь CNAME запись:
   - Type: CNAME
   - Name: cards
   - Points to: gold-oyster-258946.hostingersite.com
   
2. В Hostinger для приложения dispatcher-cards-app:
   - Добавь Custom domain: cards.dispatch4you.com

### Вариант B: Подпапка (dispatch4you.com/cards)
Это сложнее, нужно:
1. Объединить оба проекта в один репозиторий
2. Настроить роутинг

---

## 📊 3. Добавить Google Analytics (опционально)

1. Создай аккаунт Google Analytics
2. Получи Tracking ID (G-XXXXXXXXXX)
3. Добавь код в index.html перед </head>:

```html
<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX');
</script>
```

---

## 📧 4. Настроить email для домена (опционально)

В Hostinger можно создать email типа:
- info@dispatch4you.com
- support@dispatch4you.com
- admin@dispatch4you.com

Инструкция: https://www.hostinger.com/tutorials/how-to-create-an-email-account

## 🆘 Если что-то не работает:

### HTTPS не включается
- Подожди еще 12-24 часа
- Проверь что DNS записи правильные
- Проверь что CNAME файл в репозитории

### Сайт не открывается
- Проверь DNS: https://www.whatsmydns.net/#A/dispatch4you.com
- Очисти кеш браузера (Ctrl+Shift+Delete)
- Попробуй в режиме инкогнито

### SSL сертификат не выдается
- Убери Custom domain в GitHub Pages
- Подожди 5 минут
- Добавь Custom domain обратно
- Подожди еще 24 часа
