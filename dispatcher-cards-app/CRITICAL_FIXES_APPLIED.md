# 🚨 Критические исправления верстки

## Дата: 23 февраля 2026

---

## ❌ Проблемы, которые были:

1. **Desktop**: Карточки растянуты на всю ширину экрана
2. **Mobile**: Элементы перекрывают друг друга
3. **Текст**: Не вмещается и не читабелен
4. **Тема**: Переключение не работает
5. **Язык**: Смена языка не отображается
6. **Анимация**: Кнопки не запускают анимацию свайпа

---

## ✅ Применённые исправления:

### 1. Ограничение ширины (Desktop Fix)

**page.tsx:**
```tsx
<div className="relative z-10 h-full max-w-[450px] mx-auto flex flex-col">
```

- `max-w-[450px]` - строгое ограничение ширины
- `mx-auto` - центрирование по горизонтали
- Теперь на десктопе выглядит как мобильное приложение

---

### 2. Фиксированная высота карточек (Mobile Layout)

**SwipeCardStack.tsx:**
```tsx
<div className="relative flex-shrink-0 w-full h-[55vh] sm:h-[58vh] mb-3">
```

- Карточка занимает **55-58vh** (55-58% высоты экрана)
- `flex-shrink-0` - не сжимается
- Всегда остаётся место для кнопок внизу

---

### 3. Структура экрана (Flexbox Layout)

```
┌────────────────────────────┐
│ Header (flex-shrink-0)     │ ← ~50px
├────────────────────────────┤
│                            │
│ Cards (h-[55vh])           │ ← 55-58% экрана
│                            │
├────────────────────────────┤
│ Buttons (flex-shrink-0)    │ ← ~80px
└────────────────────────────┘
```

---

### 4. Компактный заголовок

**Было:** `text-2xl md:text-3xl` + подзаголовок
**Стало:** `text-lg` + одна строка

```tsx
<h1 className="flex-1 text-lg font-bold bg-gradient-to-r from-purple-400 to-pink-400 bg-clip-text text-transparent truncate">
  {t('app.title')}
</h1>
```

---

### 5. Glassmorphism для карточки

**DispatcherCard.tsx:**
```tsx
<div className="... backdrop-blur-xl border-2 border-white/20 bg-slate-900/70 ...">
```

- `backdrop-blur-xl` - размытие фона
- `bg-slate-900/70` - полупрозрачный фон (70% непрозрачности)
- Сквозь карточку просвечивает анимированный фон

---

### 6. Программная анимация свайпа

**DispatcherCard.tsx:**
```tsx
const controls = useAnimation()

useEffect(() => {
  if (triggerSwipe && isTop) {
    const targetX = triggerSwipe === 'right' ? 1000 : -1000
    controls.start({
      x: targetX,
      opacity: 0,
      transition: { duration: 0.3 }
    })
  }
}, [triggerSwipe, isTop, controls])
```

- Используется `useAnimation()` из framer-motion
- При клике на ❌ карточка улетает влево (-1000px)
- При клике на ✅ карточка улетает вправо (+1000px)
- Плавная анимация 0.3 секунды

---

### 7. Скрытие скроллбара

**globals.css:**
```css
.scrollbar-hide {
  -ms-overflow-style: none;
  scrollbar-width: none;
}

.scrollbar-hide::-webkit-scrollbar {
  display: none;
}
```

**DispatcherCard.tsx:**
```tsx
<div className="... overflow-y-auto ... scrollbar-hide">
```

- Скроллбар скрыт, но прокрутка работает
- Чистый интерфейс без визуального мусора

---

### 8. Размеры текста (Читабельность)

**Карточка:**
- Категория: `text-xs` (12px)
- Заголовок: `text-base` (16px)
- Описание: `text-sm` (14px)
- Подсказка: `text-xs` (12px)

**Кнопки:**
- Основные (❌/✅): `w-16 h-16` (64x64px)
- Информация (ℹ️): `w-14 h-14` (56x56px)
