# 🎉 Новая главная страница

## Что добавлено:

### 1. Главная страница (/)
- **Регистрация**: Поле для ввода имени
- **Выбор сложности**: 3 уровня
  - 🟢 Легкий: 15 вопросов, макс 10 ошибок
  - 🟡 Средний: 30 вопросов, макс 15 ошибок
  - 🔴 Сложный: 50 вопросов, макс 20 ошибок
- **Красивый дизайн**: Анимированный фон, градиенты

### 2. Страница теста (/test)
- Перенесено старое приложение с карточками
- Динамическое количество вопросов (15/30/50)
- Показывает имя пользователя в футере
- Показывает выбранную сложность

### 3. Улучшения
- Все шрифты увеличены на 20%
- Адаптивная логика для разных уровней
- Плавные переходы между страницами

## Как работает:

1. Пользователь открывает сайт → видит главную страницу
2. Вводит имя (или оставляет пустым = "Гость")
3. Выбирает сложность
4. Нажимает "Начать тест"
5. Переходит на /test с параметрами: `?difficulty=medium&name=Иван`
6. Проходит тест с выбранным количеством вопросов
7. После завершения может вернуться на главную

## Структура файлов:

```
src/app/
├── page.tsx          # Главная страница (регистрация + выбор)
└── test/
    └── page.tsx      # Страница теста (карточки)
```
