✅ Проект собран

⏳ Осталось загрузить на сервер

---

## 📍 ТЫ СЕЙЧАС ЗДЕСЬ (по скриншоту)

Ты на этапе выбора домена. Вот что делать дальше:

### 1️⃣ Выбери домен

**Вариант 1: Используй существующий домен**
- Если у тебя уже есть домен, введи его

**Вариант 2: Создай новый бесплатный домен**
- Нажми "Which domain extensions are free?"
- Выбери расширение из списка (.site, .online, .tech, .info и т.д.)
- Придумай название, например: `dispatcher-cards.site`

### 2️⃣ Завершить настройку в Hostinger

После выбора домена Hostinger создаст хостинг. Дождись завершения.

### 3️⃣ Загрузить файлы

**Способ 1: File Manager (рекомендуется)**

```
Hostinger Panel → Files → File Manager
↓
public_html
↓
Создать папку "cards"
↓
Загрузить все файлы из папки "out"
```

**Способ 2: FTP**

```
Скачать FileZilla
↓
Подключиться (данные FTP в Hostinger)
↓
Перейти в public_html/cards
↓
Перетащить файлы из папки "out"
```

## 🗂️ Структура файлов на сервере

После загрузки должно быть так:

```
public_html/
└── cards/
    ├── index.html
    ├── _next/
    │   ├── static/
    │   └── ...
    └── другие файлы из папки out
```

## 🌐 Открыть сайт

После загрузки открой в браузере:

```
https://твой-домен.com/cards
```

Например:
- `https://dispatcher-cards.site/cards`
- `https://mydomain.com/cards`

1. **Выбери домен** на текущем экране Hostinger

6. **Загрузи все файлы** из папки `out` (она уже готова!)

## 🆘 Нужна помощь?

**Не знаешь, как открыть File Manager?**
- В панели Hostinger найди раздел "Files" или "Файлы"
- Нажми "File Manager" или "Файловый менеджер"

**Не знаешь данные FTP?**
- В панели Hostinger: Files → FTP Accounts
- Там будут логин, пароль и адрес сервера

**Файлы загружены, но сайт не работает?**
- Проверь, что файлы в правильной папке
- Убедись, что `index.html` есть
- Попробуй очистить кэш браузера (Ctrl+F5)
