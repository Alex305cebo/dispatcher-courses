📋 ЧТО ДЕЛАТЬ ДАЛЬШЕ:

1. Выбери домен в Hostinger (ты сейчас на этом шаге)
   
2. Дождись создания хостинга (1-2 минуты)

3. Открой File Manager в панели Hostinger:
   Hostinger → Files → File Manager

4. Перейди в папку: public_html

5. Создай папку: cards

6. Загрузи ВСЕ файлы из папки "out" в "cards"

7. Открой в браузере: https://твой-домен.com/cards
