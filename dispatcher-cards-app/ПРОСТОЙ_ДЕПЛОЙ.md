# 🚀 Простая инструкция по деплою на Hostinger

## ✅ Шаг 1: Сборка завершена!

Проект уже собран. Папка `out` содержит готовые файлы для загрузки.

## 📤 Шаг 2: Загрузка на Hostinger

### Вариант A: Через File Manager Hostinger (Проще всего)

1. Войди в панель управления Hostinger
2. Найди раздел **"Files"** → **"File Manager"**
3. Перейди в папку `public_html`
4. Создай папку `cards` (если её нет)
5. Открой папку `cards`
6. Нажми **"Upload"** и загрузи ВСЕ файлы из папки `out`

### Вариант B: Через FTP (FileZilla)

1. Скачай FileZilla: https://filezilla-project.org/
2. Подключись к Hostinger (данные FTP найдешь в панели Hostinger)
3. Слева - твой компьютер, справа - сервер
4. На сервере перейди в `public_html/cards`
5. Перетащи все файлы из папки `out` на сервер

## 🌐 Шаг 3: Выбор домена

На скриншоте ты выбираешь домен. Рекомендую:

**Если у тебя уже есть домен:**
- Используй его (например: `mydomain.com`)
- Приложение будет доступно по адресу: `https://mydomain.com/cards`

**Если нужен новый бесплатный домен:**
- Выбери из списка бесплатных расширений (.site, .online, .tech и т.д.)
- Например: `dispatcher-cards.site`
- Приложение будет: `https://dispatcher-cards.site/cards`

## ✨ Шаг 4: Проверка

После загрузки файлов открой в браузере:
```
https://твой-домен.com/cards
```

Или если домен в корне:
```
https://твой-домен.com
```

## 🔧 Если нужно изменить путь

Если хочешь, чтобы приложение было в корне (без `/cards`):

1. Загрузи файлы из `out` прямо в `public_html` (не в подпапку)
2. Измени `next.config.js` - убери строку `basePath: '/cards'`
3. Пересобери: `npm run build`
4. Загрузи заново

## 📁 Что загружать

Загружай ВСЁ из папки `out`:
- index.html
- _next/ (папка)
- cards/ (если есть)
- все остальные файлы

## ❓ Проблемы?

**Страница не открывается:**
- Проверь, что файлы в правильной папке
- Убедись, что index.html есть в корне загруженных файлов

**Стили не работают:**
- Проверь, что папка `_next` загружена полностью
- Очисти кэш браузера (Ctrl+F5)

**404 ошибка:**
- Проверь путь: если файлы в `public_html/cards`, открывай `/cards`
- Если в `public_html` - открывай корень домена
