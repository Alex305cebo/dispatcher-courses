# 📱 Исправления мобильной верстки

## Дата: 23 февраля 2026

### 🎯 Проблемы, которые были исправлены:

1. **Заголовок слишком большой** - занимал 30% экрана
2. **Кнопки перекрывали карточку** - не было фиксированной структуры
3. **Интерфейс "разломанный"** - элементы налезали друг на друга

---

## ✅ Применённые исправления:

### 1. Компактный Header (`page.tsx`)

**Было:**
- Большой заголовок `text-2xl md:text-3xl`
- Отдельная строка для подзаголовка
- Крупные кнопки переключателей

**Стало:**
- Компактный заголовок `text-base sm:text-lg md:text-xl`
- Всё в одну строку: кнопка "Назад" + заголовок + переключатели
- Маленькие кнопки `w-8 h-8 sm:w-9 sm:h-9`
- Минимальные отступы `px-3 py-2`

```tsx
<header className="flex-shrink-0 px-3 py-2 sm:px-4 sm:py-3">
  <div className="flex items-center justify-between gap-2">
    {/* Кнопка назад + заголовок */}
    {/* Переключатели */}
  </div>
</header>
```

---

### 2. Фиксированная структура экрана (`page.tsx`)

**Использована Flexbox структура:**

```tsx
<div className="relative z-10 h-full flex flex-col">
  {/* Header - flex-shrink-0 */}
  <header className="flex-shrink-0">...</header>
  
  {/* Cards - flex-1 (занимает всё свободное место) */}
  <div className="flex-1 relative min-h-0">...</div>
</div>
```

**Ключевые моменты:**
- `flex-shrink-0` для header - не сжимается
- `flex-1` для карточек - занимает всё доступное пространство
- `min-h-0` - позволяет карточкам корректно масштабироваться

---

### 3. Кнопки внизу (`SwipeCardStack.tsx`)

**Структура:**

```tsx
<div className="h-full flex flex-col">
  {/* Progress - flex-shrink-0 */}
  <div className="flex-shrink-0">...</div>
  
  {/* Cards - flex-1 */}
  <div className="flex-1 relative min-h-0">...</div>
  
  {/* Buttons - flex-shrink-0 (фиксированы внизу) */}
  <div className="flex-shrink-0 py-2 sm:py-3">...</div>
</div>
```

**Изменения:**
- Кнопки теперь `flex-shrink-0` - всегда внизу
- Карточки `flex-1` - автоматически подстраиваются
- Уменьшены отступы между элементами

---

### 4. Компактная карточка (`DispatcherCard.tsx`)

**Уменьшены размеры:**
- Шрифты: `text-sm sm:text-base md:text-lg` (было `text-base sm:text-lg md:text-xl`)
- Отступы: `p-4 sm:p-5` (было `p-4 sm:p-5 md:p-6`)
- Скругления: `rounded-xl sm:rounded-2xl` (было `rounded-2xl md:rounded-3xl`)
- Индикаторы свайпа: `w-12 h-12 sm:w-16 sm:h-16` (было `w-14 h-14 sm:w-16 sm:h-16 md:w-20 md:h-20`)

**Добавлено:**
- `pointer-events-none` для glow эффекта - не мешает кликам
- `pr-1` для текста описания - отступ для скроллбара
- `leading-snug` для заголовка - компактнее

---

### 5. Компактные переключатели

**ThemeToggle:**
- Размер: `w-8 h-8 sm:w-9 sm:h-9` (было `w-12 h-12 sm:w-14 sm:h-14`)
- Форма: `rounded-lg` (было `rounded-full`)
- Иконка: `w-4 h-4 sm:w-5 sm:h-5` (было `w-5 h-5 sm:w-6 sm:h-6`)

**LanguageSwitcher:**
- Отступы: `px-2 py-1 sm:px-2.5 sm:py-1.5` (было `px-3 py-2 sm:px-4 sm:py-2`)
- Шрифт: `text-xs sm:text-sm` (было `text-sm sm:text-base`)
- Убрана стрелка вниз для экономии места

## 🎨 Результат:

✅ Header занимает ~5-7% экрана (было ~30%)

✅ Всё читаемо на мобильных устройствах
✅ Интерфейс не "разламывается" на разных размерах экрана
