📋 ЧТО ДЕЛАТЬ:

1. Загрузи проект на GitHub:
   - Открой GitHub Desktop или VS Code
   - Создай новый репозиторий "dispatcher-cards"
   - Загрузи папку dispatcher-cards-app

2. Открой Hostinger:
   - Websites → Add Website
   - Выбери "Node.js Web App"
   - Выбери "Import Git Repository"

3. Подключи GitHub:
   - Авторизуй Hostinger
   - Выбери репозиторий dispatcher-cards
   - Нажми Deploy

4. Дождись сборки (2-5 минут)

5. Готово! Приложение работает! 🎉

═══════════════════════════════════════════════════════════

🔧 НАСТРОЙКИ СБОРКИ:

Framework: Next.js
Build Command: npm run build
Start Command: npm start
Node Version: 20.x
