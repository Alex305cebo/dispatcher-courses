# Инструкция по размещению на dispatch4you.com/cards

## Шаг 1: Подключение к Hostinger

1. Зайдите на https://hpanel.hostinger.com
2. Войдите в свой аккаунт
3. Выберите домен **dispatch4you.com**

## Шаг 2: Создание папки cards

1. Откройте **File Manager** (Файловый менеджер)
2. Перейдите в папку `public_html` (корневая папка сайта)
3. Создайте новую папку с именем `cards`

## Шаг 3: Загрузка файлов

Загрузите все файлы из папки `dispatcher-cards-app/out/` в папку `public_html/cards/`:

```
public_html/
  └── cards/
      ├── _next/          (вся папка)
      ├── sounds/         (вся папка)
      ├── 404.html
      ├── index.html
      ├── index.txt
      └── test.html
```

### Способы загрузки:

**Вариант А: Через File Manager (веб-интерфейс)**
1. Откройте папку `cards`
2. Нажмите кнопку **Upload Files**
3. Выберите все файлы из `dispatcher-cards-app/out/`
4. Дождитесь завершения загрузки

**Вариант Б: Через FTP (быстрее для больших файлов)**
1. Скачайте FileZilla: https://filezilla-project.org/
2. Подключитесь к FTP:
   - Host: ftp.dispatch4you.com
   - Username: (ваш FTP логин из Hostinger)
   - Password: (ваш FTP пароль)
   - Port: 21
3. Перетащите все файлы из `dispatcher-cards-app/out/` в папку `public_html/cards/`

## Шаг 4: Проверка

После загрузки откройте в браузере:
- https://dispatch4you.com/cards/

Должно открыться приложение с карточками!

## Шаг 5: Тестовая страница (опционально)

Также доступна тестовая страница:
- https://dispatch4you.com/cards/test.html

## Возможные проблемы

### Проблема: 404 ошибка
**Решение:** Убедитесь, что файлы загружены в правильную папку `public_html/cards/`

### Проблема: Не загружаются стили
**Решение:** Убедитесь, что папка `_next` загружена полностью со всеми подпапками

### Проблема: Не работают звуки
**Решение:** Убедитесь, что папка `sounds` загружена полностью

## Обновление приложения

Когда нужно обновить приложение:
1. Пересоберите проект: `npm run build` в папке `dispatcher-cards-app`
2. Удалите старые файлы из `public_html/cards/`
3. Загрузите новые файлы из `dispatcher-cards-app/out/`

## Альтернатива: Автоматическое развертывание

Для автоматизации можно настроить:
- GitHub Actions для автоматической загрузки при push
- FTP скрипт для быстрого обновления
