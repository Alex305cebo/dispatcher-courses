# 📝 Инструкция по созданию новых страниц

## 🎯 Важно! Все новые страницы должны соответствовать единому стилю проекта

### ✅ Обязательные требования:

#### 1. **Использовать шаблон TEMPLATE_NEW_PAGE.html**
   - Скопируйте файл `TEMPLATE_NEW_PAGE.html`
   - Переименуйте в нужное имя (например: `new-feature.html`)
   - Разместите в папке `pages/`

#### 2. **Единый стиль и размеры:**
   - ✅ Фон: `linear-gradient(135deg, #667eea 0%, #764ba2 100%)`
   - ✅ Контейнер: `max-width: 1050px` (уменьшено на 25%)
   - ✅ Карточки: `border-radius: 20px`, `box-shadow: 0 20px 60px rgba(0,0,0,0.3)`
   - ✅ Кнопки: `border-radius: 50px` (pill-style), `text-transform: uppercase`, `letter-spacing: 0.5px`
   - ✅ Отступы body: `padding: 30px 20px`
   - ✅ Анимация фона: `backgroundMove` (уже в шаблоне)
   - ✅ Backdrop-filter: `blur(10px)` на всех карточках

#### 3. **Глобальная навигация:**
   Навигация уже включена в шаблон с правильными стилями:
   - 🏠 Главная (nav-btn-secondary)
   - 📚 Модули (nav-btn-success)
   - 📋 Тесты (nav-btn-warning)
   - 🎮 Симулятор (nav-btn-primary)
   - 💼 Кейсы (nav-btn-info)

#### 4. **Добавление новой кнопки в навигацию:**

**ШАГ 1:** Добавьте кнопку в вашу новую страницу:
```html
<div class="nav-links">
    <a href="../index.html" class="nav-btn nav-btn-secondary">🏠 Главная</a>
    <a href="modules.html" class="nav-btn nav-btn-success">📚 Модули</a>
    <a href="testing.html" class="nav-btn nav-btn-warning">📋 Тесты</a>
    <a href="simulator.html" class="nav-btn nav-btn-primary">🎮 Симулятор</a>
    <a href="cases.html" class="nav-btn nav-btn-info">💼 Кейсы</a>
    <!-- НОВАЯ КНОПКА -->
    <a href="new-page.html" class="nav-btn nav-btn-danger">🆕 Новая страница</a>
</div>
```

**ШАГ 2:** Добавьте стиль для новой кнопки (если нужен новый цвет):
```css
.nav-btn-info {
    background: linear-gradient(135deg, #8b5cf6, #7c3aed);
    color: white;
    border-color: rgba(255, 255, 255, 0.3);
}
.nav-btn-info:hover {
    transform: translateY(-4px) scale(1.05);
    box-shadow: 0 8px 25px rgba(139, 92, 246, 0.5);
}
```

**ШАГ 3:** Обновите навигацию на ВСЕХ существующих страницах:
- index.html
- pages/modules.html
- pages/testing.html
- pages/simulator.html
- pages/module-1.html до module-10.html
- pages/test-1.html до test-10.html
- pages/documentation.html
- pages/loadboard.html

### 🎨 Структура новой страницы:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <!-- Meta теги -->
    <title>Название | Курсы - Диспетчера</title>
    
    <!-- Стили страницы -->
    <style>
        /* Базовые стили из шаблона */
    </style>
</head>
<body>
    <!-- НАВИГАЦИЯ (обязательно!) -->
    <nav class="global-nav">...</nav>
    
    <div class="container">
        <!-- HEADER -->
        <div class="header">
            <h1>Заголовок</h1>
        </div>
        
        <!-- КОНТЕНТ -->
        <div class="content-card">
            <h2 class="section-title">Раздел</h2>
            <div class="section-content">
                <p>Контент...</p>
            </div>
        </div>
    </div>
</body>
</html>
```

### ⚠️ Важные правила:

1. **НЕ ИЗМЕНЯЙТЕ** базовые стили (фон, размеры, отступы)
2. **ВСЕГДА ДОБАВЛЯЙТЕ** глобальную навигацию
3. **ОБНОВЛЯЙТЕ** навигацию на всех страницах при добавлении новой
4. **ИСПОЛЬЗУЙТЕ** единые классы: `.container`, `.header`, `.content-card`
5. **СОХРАНЯЙТЕ** адаптивность для мобильных устройств

### 📱 Адаптивность:

Шаблон уже включает адаптивные стили для мобильных:
```css
@media (max-width: 768px) {
    body { padding: 20px 15px; }
    .header, .content-card { padding: 25px 20px; }
    .header h1 { font-size: 28px; }
    .nav-btn { font-size: 13px; padding: 12px 22px; }
}
```
