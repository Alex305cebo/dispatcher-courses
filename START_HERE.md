# 🚀 Быстрый старт

## Шаг 1: Установка Node.js

Если у вас еще не установлен Node.js, скачайте его с [nodejs.org](https://nodejs.org/)

Проверьте установку:
```bash
node --version
npm --version
```

## Шаг 2: Установка зависимостей

Откройте терминал в папке проекта и выполните:

```bash
npm install
```

Это установит все необходимые пакеты:
- express (веб-сервер)
- sqlite3 (база данных)
- bcryptjs (хеширование паролей)
- jsonwebtoken (JWT токены)
- cors (для работы с frontend)

## Шаг 3: Запуск сервера

```bash
npm start
```

Вы увидите сообщение:
```
Server running on http://localhost:3000
Backend система запущена!
Connected to SQLite database
Users table ready
```

## Шаг 4: Открытие сайта

Откройте в браузере:
```
http://localhost:3000/index.html
```

## ✅ Готово!

Теперь вы можете:
1. Зарегистрироваться на сайте (register.html)
2. Войти в систему (login.html)
3. Пользоваться всеми функциями сайта

## 🔧 Полезные команды

### Остановить сервер
Нажмите `Ctrl + C` в терминале

### Перезапустить сервер
```bash
npm start
```

### Запуск в режиме разработки (с автоперезагрузкой)
```bash
npm run dev
```

## ❓ Проблемы?

### Порт 3000 занят
Измените PORT в файле `server.js` на другой (например, 3001)

### Ошибка "Cannot find module"
Выполните: `npm install`

### База данных не создается
Проверьте права на запись в папке проекта
