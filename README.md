# Курсы - Диспетчера (Course Dispatcher)

Интерактивное приложение для обучения диспетчерской деятельности в транспортной логистике США с полноценным симулятором распределения грузов.

## 📱 Мобильное приложение

Приложение готово для установки на Android и iOS устройства!

### 🚀 Быстрый старт (PWA)

1. Разместите приложение на HTTPS хостинге (Vercel/Netlify)
2. Откройте в браузере на мобильном устройстве
3. Установите как приложение

**Подробные инструкции**: См. `MOBILE_QUICKSTART.md`

# Установите Capacitor
npm install @capacitor/core @capacitor/cli

# Инициализируйте

## 📋 Описание

Приложение представляет собой обучающий тренажёр диспетчера с практическими модулями:

- **Главная страница**: информация о курсе, счётчик загруженных грузов, выбор языка (RU/EN), кнопка оплаты.
- **Симулятор диспетчера**: пошаговый процесс работы диспетчера (4 шага).
- **Load Board**: статический реестр грузов со сложными фильтрами (по городам, штатам, времени, типам оборудования).

## 🎮 Функции и маршруты

### Главная страница (`/`)
- Информация о курсе с описанием модулей
- Счётчик grузов (из `mockLoads`)
- Переключатель языка (RU/EN) — сохраняется в `localStorage`
- Кнопка оплаты в шапке (редирект на `/checkout`)
- Две основные кнопки:
  - **Приступить к работе** → переход в симулятор (`/simulator`)
  - **Симулятор диспетчера грузоперевозок** (заголовок-кнопка) → переход в симулятор

### Симулятор диспетчера (`/simulator`)
Четырёхэтапный рабочий процесс:

1. **Подготовка водителя**
   - Ввод имени (автогенерация, если не указано)
   - Флаг готовности трака
   - Время освобождения трейлера

2. **Поиск грузов**
   - Поиск по городу/штату
   - Выбор груза из отфильтрованного списка

3. **Проверка готовности**
   - Сверка времени трейлера и погрузки
   - Автоматическое назначение водителя на груз (без alert)
   - Кнопка меняет статус на "Назначено"

4. **Связь с брокером**
   - Корректировка расстояния до погрузки, местоположение водителя, тип трейлера, вес
   - Предпросмотр сообщения брокеру в профессиональном формате
   - Кнопка **Скопировать сообщение** → копирует текст в буфер обмена
   - Кнопка **Отправить брокеру** → сохраняет в журнал (эмуляция)
   - Журнал отправленных сообщений

### Load Board (`/dispatcher`)
- Полный реестр 63+ грузов со всех штатов США
- Расширенные фильтры:
  - По городам (поиск)
  - По штатам (dropdown)
  - По типу оборудования (53ft Dry Van, Reefer, Flatbed, Amazon)
  - По времени погрузки (6 периодов: 00-04, 04-08, 08-12, 12-16, 16-20, 20-24)
- Детальная карточка груза с расчётом К/м показателя
- Автогенерация грузов для отсутствующих штатов
- Связывание грузов через `linkedTo` (кольцевая цепь)

## 📝 Данные модели (mockLoads)

Каждый груз содержит:
```typescript
{
  id: number
  shipper: string
  pickupLocation: string
  pickupCity: string
  pickupState: string
  deliveryLocation: string
  deliveryCity: string
  deliveryState: string
  distance: number // миль
  weight: number // фунты
  cargoType: string
  rate: number // доллары
  equipment: string
  createdAt: string // дата
  status: 'Available' | 'Assigned' | 'In Transit' | 'Delivered'
  assignedDriver?: string
  unloadTime: number // часы
  pickupTime: string // HH:MM
  linkedTo?: number // ID следующего груза
}
```

## 📖 Использование

1. **Главная страница**: прочитайте описание курса, используйте переключатель языка.
2. **Откройте симулятор**: нажмите кнопку "Приступить к работе" или на заголовок "Симулятор диспетчера".
3. **Пройдите 4 шага**: заполняйте поля, выбирайте грузы, проверяйте готовность, готовьте сообщение брокеру.
4. **Load Board**: альтернатива для просмотра реестра грузов и фильтрации (ссылка в симуляторе).

## 🐛 Известные ограничения

- Оплата (Stripe): требует настройки API ключей для реального использования; без них работает эмуляция.
- Сохранение: данные не сохраняются в БД; стирают при перезагрузке страницы.
- Сценарии: модуль `scenarios.ts` зарезервирован для будущего расширения (3 обучающих сценария).
