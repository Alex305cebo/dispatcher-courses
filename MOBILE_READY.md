## 🚀 Как использовать на мобильных:

### Android:
1. Откройте `android-preview.html` в Chrome
2. Нажмите меню (⋮) → "Установить приложение"
3. Приложение появится на главном экране

### iOS:
1. Откройте `ios-preview.html` в Safari
2. Нажмите кнопку "Поделиться" 
3. Выберите "На экран Домой"
4. Приложение появится на главном экране

### Прямой доступ:
- Откройте `index.html` на мобильном устройстве
- Все страницы автоматически адаптируются под размер экрана
