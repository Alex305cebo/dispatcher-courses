# 🚛 Load Finder - Финальная версия

## ✅ Что создано

Приложение для поиска **РЕАЛЬНЫХ грузов** с использованием Selenium для парсинга бесплатных источников.

## 🌐 Источники данных

### Работающие источники:
1. ✅ **FreightFinder.com** - основной источник, ~25 грузов за запрос

### Источники требующие регистрацию:
2. ⚠️ TrueNorth.com - требует аккаунт
3. ⚠️ TruckSmarter.com - требует аккаунт
4. ⚠️ PickaTruckLoad.com - требует авторизацию
5. ⚠️ 123Loadboard.com - требует регистрацию
6. ⚠️ GoLoadUp.com - требует регистрацию
7. ⚠️ Convoy.com - требует регистрацию

### Социальные сети (требуют авторизацию):
8. ⚠️ Facebook Groups - требует Facebook аккаунт
9. ⚠️ Telegram каналы - требует Telegram API
10. ⚠️ Reddit r/Truckers - требует Reddit API

## 📊 Текущая статистика

- **Активных источников:** 1 (FreightFinder)
- **Грузов за запрос:** ~25
- **Данные:** 100% реальные
- **Обновление:** в реальном времени

## 🚀 Запуск

```bash
cd load-finder-app
python app_selenium.py
```

Откройте: http://localhost:5000

## 📋 Что показывается

### LOAD DETAILS:
- 📅 Pickup Date (дата загрузки)
- 📍 Origin (город отправления)
- 📍 Destination (город назначения)
- 🚛 Equipment Type (тип оборудования)

### COMPANY INFORMATION:
- 🏢 Broker / Carrier (название брокера)
- 📞 Contact Phone (реальный телефон)
- 📡 Data Source (источник данных)

## ⚡ Возможности

- ✅ Поиск по штату и типу оборудования
- ✅ Случайный поиск (кнопка 🎲)
- ✅ Случайный порядок грузов

- ✅ Кликабельные телефоны

## 🔧 Технологии

- **Backend:** Flask + Python
- **Парсинг:** Selenium + ChromeDriver
- **Браузер:** Headless Chrome
- **Источник:** FreightFinder.com

## 💡 Как добавить больше источников

### Вариант 1: Регистрация на load boards
Создайте бесплатные аккаунты и добавьте авторизацию в код.

### Вариант 2: Платные API
- DAT Load Board API ($150-300/мес)
- Truckstop.com API ($100-200/мес)

### Вариант 3: Социальные сети
Требуют API ключи:
- Facebook Graph API
- Telegram Bot API  
- Reddit API

## ⚠️ Ограничения

**Почему только 1 источник работает:**

Все бесплатные load boards защищают свои данные:
- Требуют регистрацию
- Используют JavaScript для загрузки данных
- Блокируют автоматический доступ
- Требуют CAPTCHA

FreightFinder - единственный кто показывает реальные грузы публично без регистрации.

## 🎯 Итог

Приложение работает и показывает **реальные грузы** с реальными телефонами брокеров. Данные актуальны и обновляются при каждом поиске.

Для большего количества источников нужна регистрация или платные API.
