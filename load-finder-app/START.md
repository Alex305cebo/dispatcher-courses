# 🚀 Быстрый старт Load Finder

## Шаг 1: Установка Python

Убедитесь, что Python 3.8+ установлен:
```bash
python --version
```

## Шаг 2: Установка зависимостей

```bash
cd load-finder-app
pip install -r requirements.txt
```

## Шаг 3: Запуск приложения

```bash
python app.py
```

## Шаг 4: Открыть в браузере

Откройте: **http://localhost:5000**

## Использование

1. Введите город отправления (например: Chicago)
2. Выберите тип трейлера
3. Укажите минимальную ставку $/миля
4. Нажмите "Найти грузы"
