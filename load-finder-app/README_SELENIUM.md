# 🚛 Load Finder - Реальные грузы с FreightFinder.com

## ✅ Что работает

Приложение использует **Selenium** для извлечения **РЕАЛЬНЫХ грузов** с FreightFinder.com:

- ✅ Реальные города отправления и назначения
- ✅ Реальные названия брокеров (ALLEN LUND COMPANY, DSV Road и т.д.)
- ✅ Реальные телефоны брокеров
- ✅ Актуальные даты
- ✅ Типы оборудования (Van, Flatbed, Reefer)

## 🚀 Запуск

### 1. Установка зависимостей

```bash
pip install flask selenium webdriver-manager beautifulsoup4
```

### 2. Запуск сервера

```bash
python app_selenium.py
```

### 3. Открыть в браузере

```
http://localhost:5000
```

## 📋 Как это работает

1. **Пользователь выбирает**:
   - Штат (AZ, CA, TX, FL и т.д.)
   - Тип трейлера (Dry Van, Reefer, Flatbed)

2. **Selenium**:
   - Открывает FreightFinder.com
   - Заполняет форму поиска
   - Ждет результатов
   - Извлекает данные грузов

3. **Приложение показывает**:
   - Список реальных грузов
   - Контакты брокеров
   - Маршруты
   - Даты

## ⚠️ Важно

- **Скорость**: Selenium медленнее обычного парсинга (3-5 секунд на запрос)
- **Браузер**: Работает в headless режиме (невидимый)
- **Chrome**: Требуется Chrome/Chromium (устанавливается автоматически)
