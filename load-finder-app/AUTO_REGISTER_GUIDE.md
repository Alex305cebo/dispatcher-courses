# 🤖 Автоматическая регистрация на TruckerPath

## 🚀 Быстрый старт

```bash
cd load-finder-app
python auto_register_truckerpath.py
```

## 📋 Что делает скрипт:

1. ✅ Открывает https://temp-mail.org/ и генерирует временный email
2. ✅ Открывает https://truckerpath.com/ и заполняет форму регистрации
3. ✅ Автоматически проверяет почту каждые 5 секунд
4. ✅ Находит письмо с подтверждением
5. ✅ Извлекает ссылку для подтверждения
6. ✅ Открывает ссылку и подтверждает email
7. ✅ Сохраняет credentials в `credentials.json`

## ⏱️ Время выполнения: ~2-3 минуты

## ⚠️ Важные замечания:

### Временный email:
- ⏱️ Живет ~1 час
- ⚠️ После этого доступ к почте будет потерян
- ✅ Но аккаунт TruckerPath останется активным
- ✅ Credentials сохранены и будут работать

### Если что-то пошло не так:

**Проблема:** "Кнопка Sign Up не найдена"
**Решение:** Скрипт попробует прямую ссылку /signup

**Проблема:** "Не получено письмо"
**Решение:** 
- Проверьте интернет соединение
- Попробуйте запустить снова
- TruckerPath может задерживать отправку письма

**Проблема:** "Email уже зарегистрирован"
**Решение:** Запустите скрипт снова - будет создан новый email

**Проблема:** "Требуется CAPTCHA"
**Решение:** 
- Скрипт откроет видимый браузер
- Решите CAPTCHA вручную
- Скрипт продолжит автоматически
