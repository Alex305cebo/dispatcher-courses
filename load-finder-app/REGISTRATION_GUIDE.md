# 🤖 Автоматическая регистрация на TruckerPath

## 🎯 Процесс регистрации

Скрипт автоматически выполняет следующие шаги:

### 1️⃣ Генерация временного email
- Открывает https://temp-mail.org/ru/
- Получает автоматически сгенерированный email
- Пример: `abc123xyz@tempmail.com`

### 2️⃣ Заполнение формы регистрации
Открывает https://loadboard.truckerpath.com/carrier/sign-up и заполняет:
- ✅ **First Name** - случайное имя (John, Mike, David...)
- ✅ **Last Name** - случайная фамилия (Smith, Johnson...)
- ✅ **Phone** - случайный номер (555XXXXXXX)
- ✅ **Email (login)** - временный email с temp-mail.org
- ✅ **Password** - случайный пароль (10+ символов)
- ✅ **Confirm Password** - тот же пароль

### 3️⃣ Получение кода верификации
- Нажимает кнопку **GET CODE**
- TruckerPath отправляет 6-значный код на email
- Скрипт проверяет temp-mail каждые 10 секунд

### 4️⃣ Ввод кода и завершение
- Получает код из письма (например: `123456`)
- Вводит код в поле **6 Digit Verification Code**
- Нажимает кнопку **SIGN UP**

### 5️⃣ Сохранение данных
- Создает файл `credentials.json` с данными для входа
- Готово к использованию!

## 🚀 Использование

### Демонстрация (с паузами и пояснениями):
```bash
cd load-finder-app
python demo_registration.py
```

**Что вы увидите:**
- 2 браузера откроются одновременно
- Левый: temp-mail.org (для получения кода)
- Правый: TruckerPath (для регистрации)
- Каждый шаг с паузой и пояснением
- Время: ~3-5 минут

### Реальная регистрация (автоматически):
```bash
cd load-finder-app
python truckerpath_auto_register.py
```

**Что произойдет:**
- Полностью автоматический процесс
- Без пауз и пояснений
- Время: ~2-3 минуты
- Результат: `credentials.json` создан

## ⚠️ Важные замечания

### Временный email:
- ⏱️ Живет ~1 час на temp-mail.org
- ⚠️ После этого доступ к почте будет потерян
- ✅ Но аккаунт TruckerPath останется активным
- ✅ Credentials сохранены и будут работать постоянно

### Если что-то пошло не так:

**Проблема:** "Не получен код верификации"
**Решение:** 
- Проверьте интернет соединение
- Попробуйте запустить снова
- Увеличьте timeout в коде (120 → 180 секунд)

**Проблема:** "Email уже зарегистрирован"
**Решение:** 
- Запустите скрипт снова
- Будет создан новый email автоматически

**Проблема:** "Кнопка не найдена"
**Решение:** 
- TruckerPath мог изменить дизайн
- Проверьте скриншоты в папке (если включено)
- Сообщите об ошибке
